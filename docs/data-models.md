## Supabase Training Schema

**Location**: Supabase `public` schema

### Table: `training_samples`

| Column | Type | Description |
| :--- | :--- | :--- |
| `id` | `uuid` (PK) | Unique identifier (matches R2 scanId) |
| `image_url` | `text` | Signed R2 URL |
| `sku` | `text` | Bottle identifier |
| `confirmed_fill_pct`| `int` | Ground truth label |
| `label_source` | `text` | `admin_correction` \| `user_feedback` \| `admin_upload` |
| `confidence_weight` | `float` | 0.0 - 1.0 (based on validation flags) |
| `is_training_eligible`| `bool` | Filter for training scripts |
| `augmented` | `bool` | True if generated by Node.js script |

### Table: `model_versions`

| Column | Type | Description |
| :--- | :--- | :--- |
| `version` | `text` (PK) | e.g., `v1.0.4` |
| `active` | `bool` | Current production version |
| `mae_score` | `float` | Mean Absolute Error on test set |
| `sample_count` | `int` | Size of dataset at time of training |

## Data Flow Diagram

```
URL ?sku= ──► bottleRegistry.getBottleBySku() ──► BottleEntry
                                                       │
Camera ──► useCamera.capturePhoto() ──► compressImage() ──► base64
                                                               │
                            ┌──────────────────────────────────┘
                            ▼
              apiClient.analyzeBottle(sku, imageBase64)
                            │
                            ▼ (Worker)
              analyze.ts ──► LLM Provider ──► LLMResponse
                            │
                            ▼ (Client)
                      AnalysisResult
                            │
              ┌─────────────┼──────────────┐
              ▼             ▼              ▼
    volumeCalculator  nutritionCalculator  FillGauge
    (fillPct + geometry  (consumedMl +     (SVG render)
     → ml/tbsp/cups)      oilType → cal)
              │             │
              ▼             ▼
         VolumeBreakdown  NutritionResult
              │             │
              └──────┬──────┘
                     ▼
              ResultDisplay.tsx
                     │
                     ▼
              FeedbackPrompt ──► apiClient.submitFeedback()
                                        │
                                        ▼ (Worker)
                              feedback.ts ──► validateFeedback()
                                        │         ──► ValidationResult
                                        ▼
                              r2Client.updateScanWithFeedback()
```
