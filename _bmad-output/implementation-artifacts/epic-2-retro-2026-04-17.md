## Key Insights & Lessons Learned 💡

### AI Architecture Patterns
1. **Multi-Stage Pipeline:** Cheap fast filters (shape detection) before expensive AI calls optimizes cost and latency
2. **Reasoning-First Output:** Asking LLM to explain reasoning before answer improves accuracy
3. **Confidence Metadata:** Returning confidence levels enables better UX decisions (retake prompts)

### Failover Strategy
1. **Circuit Breaker Essential:** 7s timeout prevents indefinite waiting when providers are slow
2. **Provider Diversity:** Having Gemini + Groq fallback prevented single point of failure
3. **Transparent Failover:** Users don't need to know which provider was used - seamless experience

### Prompt Engineering
1. **Structured Output:** JSON schema in prompt dramatically improved parsing reliability
2. **Few-Shot Examples:** Including examples in prompt reduced edge case failures
3. **Iterative Refinement:** Prompt optimization (Story 2.8) was critical - don't skip this step
