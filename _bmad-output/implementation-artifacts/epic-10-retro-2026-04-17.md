## Challenges & Growth Areas

1. **Authentication omitted during route wiring (Story 10.2)**
   All three `/admin/model/*` endpoints shipped with no authentication check. The handler logic was correct; the `verifyAdminSession()` call was planned but dropped when wiring routes in `worker/src/index.ts`. Caught in security review and fixed. Root cause: auth is an opt-in per-route step, not a default middleware guard.

2. **Status drift — third occurrence across epics**
   Story 10.2 file header said `Status: ready-for-dev` after the story was done. Same pattern seen in Epic 7 stories 7.4, 7.6, 7.7. Epic 7 retro captured this as Action Item #2 ("story status field updated before marking done") — that action item was not applied in Epic 10. Team agreement is insufficient; this needs a workflow gate.

3. **Task checkbox misuse (Story 10.1)**
   Tasks 4.4 and 4.5 (manual device testing) were marked `[x]` complete but noted "Deferred to QA." Review caught and corrected. The rule needs to be explicit: `[x]` only when verifiably done, not when intended to be done.

4. **Epic FC action items not carried forward**
   6 action items from Epic FC (accessibility checklist, RTL testing, etc.) were not addressed in Epic 10. Epic 10 had different scope, so this is expected — but the carry-forward mechanism failed. Items are still open.

## Key Insights

- **Review is working as the last safety net — but safety nets shouldn't be first nets.** Both stories had issues that should have been prevented upstream (auth middleware, status gate).
- **Status drift is structural.** Three epics with the same issue means the process needs a gate, not a principle.
- **Epic 10 proves the sprint change process works.** Two blocker stories created, implemented, reviewed, and closed in a single day. The agility to create focused micro-epics is a project strength.
- **The `photoCaptured` state gap in 10.1** is worth noting: the CameraViewfinder component didn't have a `photoCaptured` state variable — it had to be added as part of this story. The orientation guide implementation surfaced a missing state in an upstream component.
