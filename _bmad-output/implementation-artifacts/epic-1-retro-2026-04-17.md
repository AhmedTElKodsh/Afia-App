## Key Insights & Lessons Learned 💡

### Architecture Decisions
1. **Cloudflare Stack Choice:** Excellent decision - Pages + Worker + R2 provided seamless integration
2. **PWA Approach:** `display: "browser"` (not standalone) avoided iOS PWA camera bugs
3. **AI Fallback Pattern:** Gemini → Groq fallback architecture became template for resilience

### Technical Patterns
1. **Component Structure:** Early component patterns (TypeScript, proper typing) set quality bar
2. **API Contract:** Worker API design with structured errors and validation proved extensible
3. **Offline Strategy:** CacheFirst for app shell, NetworkOnly for API routes worked perfectly

### Development Velocity
1. **Infrastructure First:** Completing Stories 1.1-1.3 before feature work prevented blockers
2. **Incremental Testing:** Camera flow tested on real devices early caught iOS issues
3. **Quick-Dev Trade-off:** Rapid development velocity came at cost of tracking discipline
