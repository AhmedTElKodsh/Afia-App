Security & Privacy

1

Origin Validation — CORS Whitelist

Only afia-oil-tracker.pages.dev + localhost allowed · All other origins rejected with 403

2

Rate Limiting — 10 req/min per IP

Sliding window in Cloudflare KV · Prevents abuse & AI API cost spikes · Returns 429 on breach

3

API Key Protection — Worker Secrets Only

Gemini & Groq keys stored as Cloudflare Worker env secrets · Never in client bundle or git history

4

Payload Guard — Reject > 4MB

Content-Length checked before processing · Prevents memory exhaustion & oversized AI payloads

5

Zero PII — No Accounts, No Identity

No names, emails, or phone numbers collected · R2 bucket is private (not web-accessible) · GDPR-safe by design

6

HTTPS Everywhere — Cloudflare Enforced

TLS 1.3 · HTTP→HTTPS redirect enforced at CDN edge · HSTS header on all responses

What Data Is Stored (R2)

  • images/{scanId}.jpg — bottle photo only

  • metadata/{scanId}.json — fill %, confidence, timestamp, device type, feedback

  • No user ID. No location. No personal data.

Feedback Validation (Layer 1 Sanity)

  • too_fast: response < 3s (random tap)

  • boundary_value: estimate = 0% or 100%

  • contradictory: "too low" but estimate < AI

  • extreme_delta: > 30% difference from AI

Post-POC Additions

  • User auth: JWT tokens, secure session management

  • Admin dashboard: role-based access (Phase 3)

  • R2 bucket encryption at rest (enabled by default)
