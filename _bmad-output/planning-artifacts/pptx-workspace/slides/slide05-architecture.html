System Architecture

1

User Device — React 19 PWA

QR scan opens app · Camera captures 800px JPEG (~70KB) · Local volume/nutrition logic (no server round-trips)

2

Cloudflare Worker — Edge Proxy

Origin validation · Rate limit 10 req/min/IP · Payload guard <4MB · Routes to AI providers · Stores data

3

AI Vision Providers

Primary: Gemini 2.5 Flash · JSON mode · 500 req/day free · Fallback: Groq + Llama 4 Scout (online only, POC) → Fine-tuned model (post-POC)

4

Cloudflare R2 — Object Storage

images/{scanId}.jpg + metadata/{scanId}.json · 10 GB free · ~140K scans · $0 egress · Training data

⚠ Requires credit card activation — deferred to post-POC

Request Flow

POST /analyze

→ validate → AI call → store → respond

POST /feedback

→ sanity check → update metadata

Why This Architecture

✓ API keys never in browser

✓ Zero cold starts (V8 isolates)

✓ No DB server to manage

✓ Edge CDN globally fast

✓ R2 native binding (no HTTP)

Serverless · No infra to manage
